# Werken-De-Clercq
Minimalistische terminal-style webapp voor ontwerpfiles (design services) van Werken De Clercq / DDC9000. Gebaseerd op producten van zwartopwit.be/az-list. Verkoopt geen drukwerk, maar kant-en-klare ontwerpbestanden die compatibel zijn met ZwartOpWit print services. Klanten kunnen zelf printen (goedkoper) of full-service aanvragen.
